

These scripts reproduce the experiments desribed in the article:

M. Uecker and M. Lustig, Estimating Absolute-Phase Maps Using ESPIRiT
and Virtual Conjugate Coils,


They require the Berkeley Advanced Reconstruction Toolbox
(version 0.2.07) [1]. The software is available on GitHub [2].
