

These scripts reproduce the experiments described in the article:

M. Uecker and M. Lustig, Estimating Absolute-Phase Maps Using ESPIRiT
and Virtual Conjugate Coils, Magn Reson Med (in press),
arXiv:1509.03557 [cs.CV]


They require the Berkeley Advanced Reconstruction Toolbox
v0.3.00-72-gec0600c or (when released) version v0.3.01. The
latest official release is v0.3.00 [1]. Newer versions are
available on GitHub [2].





[1] Martin Uecker et al., BART: version 0.3.00 (2016)
    DOI: 10.5281/zenodo.44511

[2] https://mrirecon.github.io/bart
